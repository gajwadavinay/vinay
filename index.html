How to use this repo

  1. Put this file as index.html in the root of your repository (or in /docs if you prefer).
  2. Add the uploaded PDF file to the repository at CN_Workbook_Session_3_Part_2.pdf (same folder as this HTML or adjust path accordingly).
  3. Enable GitHub Pages (Settings → Pages → source: main branch, folder: / (root) or /docs).

Quick Git commands

# from your project folder
git init
git add index.html CN_Workbook_Session_3_Part_2.pdf
git commit -m "Add site index and workshop PDF"
git remote add origin https://github.com/your-username/your-repo.git
git push -u origin main
